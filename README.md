# Extract Contents From Url

## 3 Steps

1. Extract Picture Album from Urls
2. Extract Picture Urls from Album
3. Save Pictures from Urls

## Installation

1. Install **ruby**
2. Install **Module**

   ```bash
    gem install nokogiri open-uri
   ```

3. Create result directory in a working directory path

   ```bash
    mkdir result
   ```

4. Let Run

   ```bash
    ruby <Filename>
   ```
